Background image property


Used internal css for styling

  1. By default, the image is repeated so it covers the entire element.
  2. After setting background-repeat property to no-repeat, the image is not repeated
  3. We can also set background-repeat property to repeat the image either horizontally or vertically to fit the screen size
  4. background-position is set to center
  5. background-attachment is set to fixed
  6. Also used two background images
